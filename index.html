입학자 교육과정 INDEX

1. 시스템 안내

입학자 교육과정 관리 시스템은 각 학과별로 입학년도 기준 교육과정을 업로드하고, 학기·학년별 개설 교과목을 자동으로 조회할 수 있도록 구성된 내부 관리 시스템입니다.

2. 주요 기능

  • 전체 교육과정(입학년도별) 업로드
  • 중간 변경사항(개정 교육과정) 업로드
  • 학과·입학년도·조회년도 기준 개설 교과목 조회
  • 업로드된 모든 버전의 교육과정 이력 확인
  • 데이터 초기화 및 관리 기능

3. 사용 방법

① 엑셀 업로드

전체 교육과정 또는 변경사항 엑셀을 업로드하여 저장합니다.

② 필터 선택

학과 → 입학년도 → 조회 학년도를 선택하면 자동으로 해당 연도의 개설 과목이 표시됩니다.

③ 개설 교과목 조회

학년·학기별 교과목, 학점, 시수, 정원 등 정보를 표 형식으로 확인할 수 있습니다.

4. 페이지 바로가기

전체 교육과정 업로드

입학년도별 전체 교육과정을 업로드하여 관리합니다.

교육과정 변경사항 업로드

중간 개정, 과목 변경·폐지·신설 내용을 반영합니다.

교육과정 조회

학과 및 학년별 개설 교과목을 조회합니다.

데이터 초기화

로컬 저장소에 저장된 모든 데이터를 초기화합니다.
